# Blueprint: 럭키 로또 번호 생성기

## 개요

이 프로젝트는 사용자를 위한 현대적이고 시각적으로 매력적인 로또 번호 생성기를 만드는 것을 목표로 합니다. 사용자는 버튼 클릭 한 번으로 무작위 번호 조합을 생성하고, 이전 생성 기록을 확인하며, 다크 모드와 라이트 모드 간에 테마를 전환할 수 있습니다. 이 애플리케이션은 최신 웹 표준(HTML, CSS, JS)을 사용하여 프레임워크 없이 구축됩니다.

## 디자인 및 기능

### 1. 시각적 디자인

- **테마**: '행운'과 '기회'를 강조하는 재미있고 활기찬 테마.
- **색상 팔레트**: 시각적 계층과 명확성을 위해 생생한 그라데이션과 부드러운 그림자를 사용합니다.
- **타이포그래피**: 가독성이 뛰어난 현대적인 산세리프 글꼴('Pretendard')을 사용합니다.
- **레이아웃**: 중앙에 카드 기반 레이아웃을 사용하여 번호 생성기에 집중할 수 있도록 합니다. 모바일 반응형으로 설계되었습니다.
- **아이콘 및 효과**: 미묘한 그림자, 그라데이션, 그리고 인터랙티브한 요소에 대한 호버 효과를 포함하여 사용자 경험을 향상시킵니다.

### 2. 핵심 기능

- **번호 생성 (구현 완료)**:
    - "번호 생성하기" 버튼을 클릭하면 1부터 45까지의 숫자 중에서 겹치지 않는 6개의 숫자를 무작위로 선택합니다.
    - 생성된 숫자는 시각적으로 구분된 공 모양 요소에 표시됩니다.
    - 숫자 값에 따라 공의 색상이 달라져 시각적 구분을 돕습니다.

- **최근 생성 번호 기록 (구현 완료)**:
    - 생성된 번호 조합은 "최근 생성 번호" 목록에 추가됩니다.
    - 각 기록 항목은 생성된 시간과 번호 조합을 표시합니다.
    - 이 기록은 브라우저의 `localStorage`에 저장되어 사용자가 페이지를 새로고침해도 유지됩니다.

- **테마 변경 (구현 예정)**:
    - 사용자가 다크 모드와 라이트 모드 간에 전환할 수 있는 토글 버튼을 제공합니다.
    - 선택된 테마는 `localStorage`에 저장되어 다음 방문 시에도 유지됩니다.

## 기술 스택

- **HTML**: 시맨틱 마크업.
- **CSS**: CSS 변수, Flexbox, Grid를 사용한 현대적인 CSS.
- **JavaScript**: DOM 조작, 이벤트 리스너, `localStorage`.

## 현재 구현 계획

1.  **`index.html` 수정**:
    -   헤더에 테마 전환 버튼(아이콘 포함)을 추가합니다.
2.  **`style.css` 수정**:
    -   라이트 및 다크 모드에 대한 색상 변수를 정의합니다.
    -   테마 전환 버튼의 스타일을 지정합니다.
    -   `body`에 `.dark-mode` 클래스가 적용될 때 색상이 변경되도록 스타일을 추가합니다.
3.  **`main.js` 수정**:
    -   테마 전환 버튼에 대한 참조를 가져옵니다.
    -   버튼 클릭 시 `body` 요소에 `.dark-mode` 클래스를 토글하는 이벤트 리스너를 추가합니다.
    -   선택된 테마 설정을 `localStorage`에 저장하고 페이지 로드 시 이를 적용하는 로직을 구현합니다.
